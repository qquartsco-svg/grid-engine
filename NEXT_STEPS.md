## 2️⃣ 다음 작업 우선순위 (산업 기준)

### 🥇 1순위 — CNC/로봇 외란 복귀 데모 (핵심)

**이건 필수다. 이거 없으면 세일즈 불가.**

**목표**: "Grid Engine이 있으면 뭐가 달라지는가?"를 한 장면으로 보여주기

**데모 시나리오**:
1. 정상 궤적 실행
2. 외란 주입 (impulse / step disturbance)
3. PID only → 복귀 시간 측정
4. PID + Grid → 복귀 시간/오차 비교

**측정 지표**:
- 복귀 시간 (settling time)
- RMS 위치 오차
- 회전축(A/B) 위상 안정성

**완료 기준**:
- 하나의 그래프 + 하나의 숫자면 충분
- 실제 하드웨어 ❌ → 시뮬 + 모델 기반이면 OK

**파일 위치**: `benchmarks/recovery_test.py`

---

### 🥈 2순위 — "5D Ring = Robot Joint Memory" 명확화

**필요한 것**:
- 관절 관점 설명 자료
  - X/Y/Z = Cartesian
  - A/B = Joint / Tool orientation
- "CNC → 로봇 팔" 전환 다이어그램
  - CNC 5축 = 로봇 관절 제어의 특수 케이스

**완료 기준**: 그림 1장으로 정리

**파일 위치**: `docs/CNC_TO_ROBOT_MAPPING.md`

---

### 🥉 3순위 — ROS / Control Stack 어댑터 스텁

**질문**: "이거 ROS에 어떻게 붙이나요?"

**요구사항**:
- 완성 구현 ❌
- 인터페이스 스텁만 있으면 충분

**예시**:
```python
GridEngine -> desired_pose
desired_pose -> ros_control / MoveIt
```

**완료 기준**: 코드 20~30줄짜리 "연결 예시"면 끝
- 실제 ROS 실행 ❌ 필요 없음

**파일 위치**: `interfaces/ros_stub.py`
